class xlets_settings.SettingsBox(pages_definition=[], instance_info={}, main_app=None, xlet_meta=None)[source]¶

Bases: xlets_settings.common.BaseGrid

Settings box.

stack¶

A Gtk.Stack.

Type

Gtk.Stack

Initialization.

Parameters

  • pages_definition (list, optional) -- The list containing the data to generate all window widgets.

  • instance_info (dict, optional) -- Xlet instance information.

  • main_app (None, optional) -- The MainApplication that some widgets need to get some data from. For example, the IconChooser widget needs the list of icons from the current icon theme. Instead of generating that data every single time an IconChooserDialog is opened, it will be generated once the first time such dialogs is opened and succesive openings from any other IconChooser widget will use the same data.

  • xlet_meta (None, optional) -- Xlet metadata.

Raises

exceptions.UnkownWidgetType -- Inform that the widget type is non existent.
