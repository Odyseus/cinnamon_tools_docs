Parameters

  • title (None, optional) -- Section title.

  • subtitle (None, optional) -- Section subtitle.

  • section_info (dict, optional) -- Data used to create a button at the end of the section title that opens a dialog. The dictionary can contain two keys. And a "message" key that will store the text in valid markup that will be used on the displayed message.
