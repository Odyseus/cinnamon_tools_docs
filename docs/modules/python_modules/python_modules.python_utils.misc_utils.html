python_modules.python_utils.misc_utils.merge_dict(first, second, logger=None, extend_lists=True, append_to_lists=True)[source]¶

Merges second dictionary into first dictionary and return merged result.

It deep merges keys of type dict and list. Any other type is overwritten.

Parameters

  • first (dict) -- A dictionary to which to merge a second dictinary.

  • second (dict) -- A dictionary to merge into another dictionary.

  • logger (LogSystem) -- The logger.

  • extend_lists (bool, optional) -- When dealing with lists, extend the first list one with the second one.

  • append_to_lists (bool, optional) -- If first is a list and second isn't, append second to first.

Returns

A merged dictionary.

Return type

dict
