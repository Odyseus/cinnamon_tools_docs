python_modules.python_utils.prompts.confirm(prompt=None, response=False)[source]¶

Prompts for yes or no response from the user.

Parameters

  • prompt (None, optional) -- The prompt text.

  • response (bool, optional) -- "response" should be set to the default value assumed by the caller when user simply types ENTER.

Returns

True for "yes" or False for "no".

Return type

bool

Raises

exceptions.KeyboardInterruption -- Halt execution on Ctrl + C press.

Note

Based on: Prompt the user for confirmation (Python recipe).

Modifications:

  • Eradicated Python 2 code and added transparent handling of upper/lower case input responses.

Examples

>>> confirm(prompt='Create Directory?', response=True)
Create Directory? [Y|n]:
True
>>> confirm(prompt='Create Directory?', response=False)
Create Directory? [N|y]:
False
>>> confirm(prompt='Create Directory?', response=False)
Create Directory? [N|y]: y
True
